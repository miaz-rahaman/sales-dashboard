📊 Sales Data Analysis Dashboard

A complete end-to-end Retail Sales Analytics Project built using Python, showcasing data cleaning, exploratory data analysis (EDA), KPI tracking, and business intelligence visualizations.

This project simulates a real-world retail environment and transforms raw sales data into an interactive analytical dashboard for decision-making.

🚀 Project Overview

The Sales Data Analysis Dashboard analyzes multi-year retail sales performance across product categories to uncover:

✅ Revenue trends
✅ Seasonal demand patterns
✅ Category performance
✅ Customer return behavior
✅ Discount impact on revenue
✅ Year-over-Year business growth

The dashboard replicates how data analysts and business intelligence teams monitor company performance.

🧠 Business Objectives

Analyze retail sales performance from 2022 – 2024

Identify top-performing product categories

Detect seasonal sales trends

Measure return rates & discount effects

Generate executive-level KPI insights

Present findings through a visual dashboard

🏗️ Project Workflow
Data Generation → Data Cleaning → EDA → KPI Analysis → Visualization → Insights
1️⃣ Data Generation

Synthetic retail dataset including:

Product Categories

Units Sold

Average Price

Discounts

Revenue

Product Returns

Monthly & Yearly timestamps

Realistic business noise such as:

Missing values

Negative quantities

Invalid prices

2️⃣ Data Cleaning

Data preprocessing steps include:

Missing revenue imputation using category median

Correction of negative unit sales

Replacement of zero pricing values

Net unit calculation after returns

Ensures production-ready analytical data.

3️⃣ Exploratory Data Analysis (EDA)

Key Performance Indicators calculated:

💰 Total Revenue

📦 Units Sold

🏷️ Average Discount

🔁 Return Rate

📈 Year-over-Year Growth

🏆 Category Revenue Ranking
